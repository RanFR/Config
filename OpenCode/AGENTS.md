# Instruction

## Language Requirements

- Always respond in **Simplified Chinese**
- Write code comments and documentation in **English**

## Discussion First

Before implementation, confirm the requirements and solutions to avoid blind operation.

- **No blind execution:** Before writing code or performing critical operations, discuss with me to confirm requirement details and technical approaches.
- **Solution confirmation:** Only begin execution after we reach consensus on the solution. If requirements are unclear, ask questions proactively—don't guess.

## Task Management

Use TodoWrite proactively when the task has 3+ distinct steps or involves non-trivial multi-step work. Break down complex tasks into a todo list before starting, and update status in real time as you work:

- Mark `in_progress` before starting each item (only one at a time)
- Mark `completed` immediately after finishing — do not batch completions
- Add follow-up items as they are discovered

## CLI Tools

- Use `uv` for Python tasks; if not installed, fallback to `python` and `pip`
