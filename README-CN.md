## PX4

针对 [PX4-Autopilot](https://github.com/PX4/PX4-Autopilot) 仿真环境使用。

在 PX4 **v1.12.3**版本上经过测试。

修改脚本中的 _SRC_DIR_ and _BUILD_DIR_ 路径，然后按照下面的命令运行即可：

```bash
. setup_px4_autopilot.sh
```
